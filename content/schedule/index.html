The schedule will be fully built out by attendees the first 45 minutes of each day. We'll take a photo and post it here and twitter after the schedule has been created. Unconferences can be nebulous so here's an idea of what to expect!

Example / planned discussions

  • Library usage discussion - Who's using what, experiences, likes / dislikes. LitElement vs VanillaJS vs any others!
  • Theming web components - custom properties vs css variables
  • Light DOM vs shadow DOM - Pros & cons, lessons learned
  • making hello-world - Creating a hello-world element with open-wc tooling
  • Tooling workflows - Rollup? Webpack? ESbuild?
  • HAX End users - hax.psu tour, discussion and roadmap prioritization for overall project
  • A11y at scale - Scaling out web components to an organization while maintaining accessibility and performance
  • Vercel / Microfrontend development - Project show cases, code deep dives, planning for the future
  • Bundle vs unbundle - When to do one vs the other, CDN discussions
  • Anything you bring to the table! These are just suggestions based on who's going to be there!
